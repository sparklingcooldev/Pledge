# Pledge smart contract

## Installation

`yarn`

## Compile contract

`yarn build`

## Deploy to Ropsten

  - Check/edit the input params in file `migrations/1_deploy.js`
  - Create the file `.secret` containing the private key
  - Run this cmd: `yarn deploy-ropsten`
